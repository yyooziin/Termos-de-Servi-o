  <h1>Política de Privacidade</h1>
  <p>Nosso bot Discord respeita sua privacidade. Aqui explicamos como tratamos seus dados:</p>

  <h2>Dados Coletados</h2>
  <p>O bot pode coletar informações básicas necessárias para funcionamento, como:</p>
  <ul>
    <li>Nome de usuário e ID do Discord;</li>
    <li>Comandos que você usa;</li>
    <li>Dados de perfil, se você fornecer.</li>
  </ul>

  <h2>Uso dos Dados</h2>
  <p>Esses dados são usados apenas para funcionamento do bot, estatísticas internas e melhorias de experiência.</p>

  <h2>Compartilhamento</h2>
  <p>Não vendemos nem compartilhamos seus dados com terceiros fora do Discord.</p>

  <h2>Segurança</h2>
  <p>Tomamos medidas razoáveis para proteger seus dados, mas não garantimos segurança absoluta.</p>

  <p>Se tiver dúvidas, entre em contato pelo Discord.</p>
